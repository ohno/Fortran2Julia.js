# Fortran2Julia.js

Fortran2Julia.js is a tool for converting Fortran programs to Julia programs using [Groups and backreferences](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_expressions/Groups_and_backreferences). The copy button is implemented using [Interact with the clipboard](https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/Interact_with_the_clipboard).
